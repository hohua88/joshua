更改为sudo gem install -n /usr/local/bin cocoapods 还是同样错误，原因是缺少ffi 解决方案：

sudo gem install -n /usr/local/bin ffi

成功后，再次执行sudo gem install -n /usr/local/bin cocoapods即可。
